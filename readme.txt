== Description ==
Khabardigonto is a lightweight, responsive Bengali news/magazine WordPress theme. Inspired by Prothom Alo, it features a customizable homepage, dark mode, breaking news ticker, category blocks, author archive, opinion layout, and more.

== Features ==
* Breaking News Marquee
* Featured Posts & Category Blocks
* Tabbed Sidebar: Recent, Popular, Discussed
* Post Format Support (Video, Quote, Image)
* Author Bio & Archive Page
* Customizer: Logo, Breaking Toggle, Sidebar, Lazy Load
* Widgetized Footer & Newsletter Box
* RTL & Bengali Font Ready
* Mobile First & SEO Optimized

== Installation ==
1. Upload the theme zip via Appearance > Themes > Add New > Upload.
2. Activate the theme.
3. Customize from Appearance > Customize.
